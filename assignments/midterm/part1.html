# Midterm - Part 1: Information architecture

# Details

Task: Develop an Enhanced Information Architecture

Create a refined information architecture for Winterlude, incorporating insights from this week's lesson.

Essential Content Elements:

  • List of Events (schedule view, alphabetical list, sorted by category)
  • Event details
  • List of locations
  • Map view of locations
  • What's on now
  • Official hotels
  • Official restaurants
  • Shuttle information (stops, routes, schedules)
  • General information
  • Sponsors
  • Canal Services
  • Canal ice conditions

# Deliverable for this exercise:

  • Output: Information Architecture Diagram
  • Format: Jpg/png/pdf file
  • Filename Convention: FirstName_lastName_IAdiagram.ext

Note

Do not just copy the information architecture for the existing Winterlude (opens new window) website. It's not very good, and either way, that's plagiarism.
