# Week 3 - Heuristic Evaluation

# Overview

This exercise involves conducting a detailed heuristic evaluation in pairs, focusing on an existing application that you frequently use but find challenging or unsatisfactory.

# Guidelines

  1. Application Selection: Choose an application that you are familiar with and have used extensively, yet find problematic or frustrating. Ensure the app is complex enough to identify a minimum of 8 usability issues.

  2. Individual Work Option: If working alone, you are required to identify at least 4 usability issues in your chosen app.

  3. Analysis and Reporting: Move past your initial dissatisfaction with the app and provide a structured, objective critique. This involves:

    • Identifying Usability Issues: Find at least 8 elements that either adhere to or breach the usability heuristics discussed in our lectures and readings.
    • Documentation Requirements: For each issue, detail the following in the provided template:
      • Feature or application part in question
      • Description of the issue
      • Relevant heuristic(s)
      • Issue severity (refer to the provided severity scale)
      • Suggested improvements
      • Visual evidence (screenshots)
    • Focus on Content Over Form: Prioritize a thorough and objective analysis over aesthetic aspects of the document.

  4. Reference Material: Utilize the 10 Usability Heuristics for User Interface Design (opens new window) as a guideline for your evaluation.

  5. Diverse Observations: You may find multiple issues within a single feature or need to highlight different features for each observed issue.

  6. Template for Documentation: Follow the structure outlined in the provided example file: heuristic Evaluation Template

# Deliverables

  • Complete the assignment using the provided template.
  • Group submissions should include each member's name alongside their contributions.
  • File Name Format: FirstNameLastName_HeuristicEvaluation.docx

WARNING

Distinguish usability issues from bugs, feature requests, and business decisions (e.g., ad placement).
